# conventional-commits 📝

**Commits that follow the convention.**

Checks `git commit -m` messages against [Conventional Commits](https://www.conventionalcommits.org)
and blocks the ones that don't — e.g. `feat: add login`, `fix(api): handle null`. Commits
with no inline message (the editor opens) pass through.

```bash
npx agentreflex add conventional-commits
```
